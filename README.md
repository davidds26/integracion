#  Integracion Aplicación en .NET 8
Este proyecto de momento está basado en practicar git mas que sus funcionalidades

## Instalación
### 1 Paso
**1 forma**. Dar clic en Code y luego en Donwload Zip
**2 forma.** Crear una carpeta, ingresar a git bash y ejecutar

    git clone https://github.com/davidds26/integracion.git
### 2 Paso
- Crea una base de datos llamada **IntegracionDB**
- Ejecutar los scripts que se encuentra dentro de la carperta **Scripts**/**integracionDB.sql** 

## Ejecutar aplicación
### 3 Paso
- **dotnet build ** para compilar la aplicación
- **dotnet watch run** para ejecutar la aplicación

**Nota:** Asegúrate de configurar correctamente la cadena de conexión para tu entorno de trabajo o servidor, si es necesario.
